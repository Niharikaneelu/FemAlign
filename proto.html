PROTOTYPE DEVELOPMENT

Key Features

1. Backrest

Equipped with sensors to monitor posture, stress, and hydration. Auto-adjusts shape to improve posture and send wellness alerts.

2. Heating Pad Feature

Integrated heating pad beneath the armrest provides warmth to relieve back pain. Adjustable temperature for personalized comfort.

3. Login System for Employees

Employees access the chair via ID cards. Health data is securely collected and stored for company-level wellness analytics.

4. Leg Rest Feature

Adjustable leg rest to reduce lower-body strain and improve circulation. Easily managed via the chair's control panel.

5. Control Panel

Control panel on armrest manages heating pad, leg rest, alerts, and user profiles with a clear LED display.
